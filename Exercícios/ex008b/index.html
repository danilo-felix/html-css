Código-fonte / Pré-formatado

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Para representar códigos, usamos a tag CODE, que converte a fonte do código para uma monoespaçada.

Também é possível representar códigos com a identação, usando a tag PRE. 

            
    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print(f'O número {num} é PAR.)
    else:
        print(f'O número {num} é ÍMPAR.')

Citações completas

As citações completas são representadas pela tag BLOCKQUOTE. É possível e recomendável utilizar o parâmetro CITE na blockquote para sinalizar o link que hospeda o documento citado. O parâmetro será reconhecido pelo navegador e por mecanismos de busca.

Abreviações

As abreviações são representadas pela tag ABBR. Esta tag pede o parâmetro TITLE, que armazena o significado da abreviação. Isto é útil para mecanismos de busca localizarem seu site.

Texto invertido

Podemos inverter textos utilizando a tag BDO. Esta tag pede o parâmetro DIR (direction), que pode ser RTL (right to left) ou LTR (left to right), que indica como os caracteres serão expostos no site.
