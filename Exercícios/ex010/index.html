Usando links externos

Para usar links externos devemos usar a tag A. A tag A pede a referência HREF (HyperText Reference), na qual deve constar o link a ser acessado.

Para que o link externo não abra na mesma aba do seu site, mas sim numa aba nova, devemos usar o parâmetro TARGET, com indicação "_BLANK" e o parâmetro REL, com indicação "EXTERNAL"

Usando links internos

Para utilizar links internos deve-se utilizar a mesma tag A, porém a referência utilizada deve apontar para a página interna do site, com REL = "NEXT".
